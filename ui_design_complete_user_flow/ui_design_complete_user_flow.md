# Complete User Flow (Blue Light)

## Entry
- User signs in or creates account.

## Core navigation
- Home: activity feed, nearby friends, suggested friends.
- Discover: incoming friend requests + find new friends.
- Friends: friend list, profile preview, remove flow.
- Map: live location + friend markers/clustering.
- Messages: conversation list, new message, chat thread.
- Profile: username/photo/settings, location tracking, trusted contacts.

## Safety flow
- SOS from top app bar.
- Required audience selection.
- Optional situation + emergency level.
- Alert appears in recipient activity feed.
- Recipient can open Alert Details.
- Recipient can Message sender and Get Location route.

## Friendship lifecycle
- Send request -> recipient sees request -> accept/decline.
- Activity notifications for request and acceptance.
- Request cancellation supported.

## Message lifecycle
- Create chat from friend list/new message.
- Real-time thread.
- Swipe-to-delete with confirmation.

## Map/location rules
- User must have tracking enabled.
- Nearby and map visibility respect friend relationship and location settings.
