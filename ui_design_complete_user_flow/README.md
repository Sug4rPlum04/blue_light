# UI Design + Complete User Flow Submission

This folder contains the package for the assignment requirement:
"Submit a screenshot of your UI design (the generated prototype showing your complete user flow)."

- `screens/` -> put all screen captures here
- `ui_design_complete_user_flow.md` -> end-to-end flow summary
- `capture_checklist.txt` -> quick capture checklist
- `capture_flow.ps1` -> optional helper script for screenshots (if a device/emulator is connected)
